# Инструкция по использованию функций троллинга

## Загрузка файлов

### Для администратора:

1. **Войдите в админ-панель** с логином `TheWorldV` и паролем `qwerty`

2. **Выберите пользователя** из списка слева

3. **Используйте кнопки троллинга:**
   - `[ SHOW IMAGE ]` - Показать картинку на весь экран
   - `[ PLAY SOUND ]` - Воспроизвести звук
   - `[ SHAKE SCREEN ]` - Встряхнуть экран
   - `[ GLITCH EFFECT ]` - Глитч-эффект

4. **Загрузка своих файлов:**
   - Нажмите на кнопку `[ SHOW IMAGE ]` или `[ PLAY SOUND ]`
   - Откроется модальное окно с файлами
   - Нажмите `[ UPLOAD NEW FILE ]`
   - Выберите файл с компьютера
   - После загрузки файл появится в списке
   - Нажмите `[ SELECT ]` чтобы отправить его пользователю

## Поддерживаемые форматы

### Изображения:
- JPG / JPEG
- PNG
- GIF
- SVG
- WEBP

### Аудио:
- MP3
- WAV
- OGG
- M4A

## Предустановленные файлы

В папке `/public/media/images/` уже есть несколько готовых изображений:
- `hacker.svg` - Сообщение Anonymous
- `skull.svg` - Предупреждение с черепом
- `glitch.svg` - Глитч экран
- `matrix.svg` - Матрица код
- `warning.svg` - Предупреждение безопасности

## Как это работает:

1. Админ загружает файл через интерфейс
2. Файл сохраняется на сервере в `/public/media/`
3. Админ выбирает файл и отправляет пользователю
4. У пользователя:
   - Картинка появляется на весь экран (можно закрыть кликом)
   - Звук воспроизводится автоматически
   - Эффекты применяются к экрану

## Ограничения:

- Максимальный размер файла: **10 МБ**
- Файлы доступны всем (публичная папка)
- Загружать файлы может только админ

## Советы:

- Используйте небольшие файлы для быстрой загрузки
- Для звуков рекомендуется формат MP3
- Для картинок с анимацией используйте GIF
- SVG файлы занимают меньше места и хорошо масштабируются
